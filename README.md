# 💎 Dana Hemat

Aplikasi manajemen keuangan pribadi premium berbasis **Local-First** — semua data tersimpan di browser Anda.

## 🚀 Tech Stack
- **Vite + React** (JSX)
- **Tailwind CSS v4** (dark mode, glassmorphism)
- **Recharts** (AreaChart, PieChart, BarChart)
- **Dexie.js** (IndexedDB wrapper)
- **React Router v6**
- **PWA** (Service Worker + manifest.json)

## ✨ Fitur
1. PWA — bisa diinstall di HP
2. Dashboard real-time dengan grafik 7 hari
3. Analitik Pie & Bar chart
4. Smart Budgeting + Aturan 50/30/20
5. Kalkulator: Bunga Majemuk, Kredit Efektif, Inflasi
6. Form transaksi dengan auto-format Rupiah
7. Data grid + sorting + filtering
8. Manajemen kategori kustom + emoji picker
9. Export CSV
10. Dark mode (warna #121212 & #1E1E1E)

## 🔧 Cara Jalankan
```bash
npm install
npm run dev
```

## 🚀 Deploy ke Vercel
```bash
# Push ke GitHub lalu connect di vercel.com
# atau
npx vercel --prod
```
